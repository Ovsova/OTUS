<h1 align="center"> Телефонный справочник.</h1>
Запуск осуществляется при запуске main.py  
  
Имеет следующий функционал при нажатии соответвуйющих цифр:  
1 - Показать все контакты  
2 - Создать новый контакт  
3 - Найти контакт  
4 - Изменить контакт  
5 - Удалить контакт  
6 - Сохранить файл  
7 - Выход
